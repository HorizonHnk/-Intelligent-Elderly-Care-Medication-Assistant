STEP 2: Install Libraries (One-Time Setup)
═══════════════════════════════════════════════════════════════════

In Arduino IDE:
1. Tools → Manage Libraries (Ctrl+Shift+I)

2. Install these 3 libraries:

   Search: "WebSockets"
   → Install: "WebSockets by Markus Sattler"

   Search: "ArduinoJson"
   → Install: "ArduinoJson by Benoit Blanchon" (Version 6.x)

   Search: "LiquidCrystal I2C"
   → Install: "LiquidCrystal I2C by Frank de Brabander"

3. Close Library Manager

═══════════════════════════════════════════════════════════════════
STEP 3: Configure Arduino IDE
═══════════════════════════════════════════════════════════════════

A. Select Board:
   Tools → Board → ESP32 Arduino → "ESP32 Dev Module"

B. Configure Settings:
   Tools → Partition Scheme → "Minimal SPIFFS (1.9MB APP with OTA/190KB SPIFFS)"
   Tools → Upload Speed → "921600"
   Tools → CPU Frequency → "240MHz (WiFi/BT)"
   Tools → Flash Mode → "QIO"
   Tools → Flash Size → "4MB (32Mb)"

C. Select Port:
   1. Connect ESP32 to USB
   2. Tools → Port → Select your COM port (COM3, COM4, etc.)

═══════════════════════════════════════════════════════════════════

1. Click Upload button (→) or press Ctrl+U

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

❌ No COM Port shown:
   → Install CP210x or CH340 USB driver
   → Check Device Manager → Ports (COM & LPT)

❌ Upload failed:
   → Hold BOOT button while uploading
   → Try lower upload speed (115200)

❌ Won't connect to WiFi:
   → Check WiFi name is exactly "ESP32SEG"
   → Check password is exactly "letmeinplease"
   → ESP32 only supports 2.4GHz (not 5GHz) ✅ Your network is 2.4GHz

❌ Website can't connect:
   → Make sure phone/computer is on ESP32SEG network
   → Ping ESP32: ping 192.168.x.xxx
   → Test in browser: http://192.168.x.xxx/api/status
